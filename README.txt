Trabalho final de linguagem de programação I

Esse projeto visa usar conceitos de programação como alocação dinâmica, manipulação de arquivos, para criar uma simulação de um sistema de aeroporto.

o programa "airplane.c" será responsável por criar e manipular dados dos aviões 

o programa  "airportS.c" será responsável por toda manipulação de cadastro de clientes, marcação de viagens.

arquivos:
    airplaneM.txt
        - Armazena os modelos de aviões disponiveis.
    trips.txt
        - Armazena todas as viagens marcadas e horários disponiveis
    users.txt
        - Armazena os dados dos usuários cadastrados
